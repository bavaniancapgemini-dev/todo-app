# ✅ To-Do App

A modern and efficient To-Do Application built to help users organize tasks, manage daily activities, and improve productivity. This project demonstrates concepts of task management, CRUD operations, validation, data handling, and modular application architecture.

---

## 🚀 Features

- Add Taks
- Upsate Tasks
- Delete Tasks
- Mark Tasks as Completed
- Task Prioritization
- Due Date Management
- Search & Filter Tasks
- Real-Time Task Tracking
- Input Validation
- Persistent Data Storage
- Clean and Modular Architecture

---

## 🛠️ Tech Stack

- Python / JavaScript
- SQLite / MySQL
- File Handling
- Object-Oriented Programming
- JSON / Database Storage

## 📌 Module Overview

### `main.py`
Acts as the main entry point of the application.

### `tasks.py`
Handles task creation, updating, deleting, and task management operations.

### `database.py`
Responsible for database connection, queries, and secure data storage.

### `validations.py`
Contains input validation and error checking logic.

### `utils.py`
Reusable helper functions used across the project.

### `filters.py`
Handles searching, filtering, and sorting of tasks.

### `reports.py`
Generates task summaries and productivity reports.

## ⚡Installation

Clone the repository:

```bash
git clone https://github.com/to-do-app/to-do-app.git
```

Move into the project directory:

```bash
cd to-do-app
```

Run the project:

```bash
python main.py
```
